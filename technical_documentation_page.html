Your first lines of code

Is it your first time programming? Don't worry, start with something simple. The familiar 'Hello world' can be printed on the screen with a few simple lines:

print("Hello World!")

In no time you will be able to assign variables and start finding the fun side of python:

greetings = "hello there!" print(greetings) hello there!

You can ask python to identify the variables for you:

funny_number = 69 type(funny_number) int

Also, you can make your own lists:

pj_name = ['Kenobi', 'Qui-Gon', 'Mace Windu', 'Yoda']

We can also simplify the code for generating the list of random draws considerably by using something called a list comprehension.

animals = ['dog', 'cat', 'bird'] plurals = [animal + 's' for animal in animals] plurals ['dogs', 'cats', 'birds]
